# CV Application

This is a single-page application (SPA) built with React and Vite as part of the [Odin Project](https://www.theodinproject.com/). The application allows users to create, edit, and export a professional CV in PDF format.

You can view the live version of the project [here](https://dlastic-cv-generator.netlify.app).

## Features

- Structured Form Sections: Collect personal details, professional experience, education, skills, and languages.
- Data Import/Export: Save progress as JSON, reload it later, load a provided example to explore the layout and clear the form.
- PDF-Friendly Preview: Real-time CV preview that prints neatly to PDF for quick sharing.

## Technologies Used

- React + Vite: SPA scaffolding, fast dev server, and component-based UI.
- JavaScript: Application logic and interactivity.
- CSS: Styling and layout.
- Netlify: Hosting and continuous deployment.
